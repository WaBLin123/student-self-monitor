# 學生自我監控系統

純前端、單檔 `index.html` 的課堂自我監控工具。  
可用於記錄學生每節課的打卡、學習檢核與抽離活動，並支援本機保存與資料匯出。

## 功能總覽

- 動態產生 `週一～週日` 課表（固定節次：早自習到第七節，含下課、午餐/午休、掃地）
- 三種紀錄模式：
  - **打卡模式**：`簽到`、`簽退`
  - **服藥紀錄**：早自習與午餐/午休可額外按 `服藥`
  - **學習檢核**：三項 checkbox（完成課堂學習活動 / 未與同學玩鬧 / 遵守教師指令）
  - **抽離活動**：特定節次可選活動（寫評量、織圍巾、晤談、放空）
- 顯示模式切換：
  - **當天**：自動判斷今天星期幾，只顯示當天欄位
  - **當週**：顯示整週
- 匯出 CSV：
  - 當週模式匯出整週
  - 當天模式只匯出今天
- 自動本機保存（`localStorage`）：
  - 重新整理後保留紀錄
  - 保留你上次選擇的顯示模式（當天/當週）

## 視覺狀態

- `normal`：綠色（正常/完成）
- `pullout`：黃色（抽離活動）
- `warning`：紅色（學習檢核未完成）

## 使用方式

1. 直接用瀏覽器開啟 `index.html`
2. 左上角選擇要查看 `當天` 或 `當週`
3. 依節次填寫紀錄：
   - 打卡格：按 `簽到` / `簽退`（早自習、午餐/午休可按 `服藥`）
   - 學習格：勾選三項後按 `✔`
   - 抽離格：選活動後按 `✔`
4. 右上角按 `匯出資料` 下載 CSV

## 匯出欄位（CSV）

- `星期`
- `節次`
- `類型`（打卡 / 學習檢核 / 抽離活動）
- `簽到`
- `簽退`
- `服藥`
- `學習完成`
- `未玩鬧`
- `遵守教師指令`
- `活動`

## 可調整設定

可直接在 `index.html` 調整以下常數：

- `days`：顯示星期範圍
- `schedule`：節次與類型
- `selfPeriods`：哪些 `週X_節次` 要顯示抽離活動

## 技術說明

- 原生 `HTML` + `CSS` + `JavaScript`
- 不需後端與資料庫
- 使用 `localStorage` 保存資料（瀏覽器本機）

## 目前限制

- 資料只存在單一瀏覽器、單一裝置（清除瀏覽資料會遺失）
- 無帳號系統、無多人同步
- 無後台統計圖表與歷史查詢頁面
