# 學生自我監控系統（整合版）

一個純前端、單檔 HTML 的課堂行為與時間管理工具。  
可記錄學生在每日課表中的打卡、學習檢核與自主活動，並提供即時回饋、簡易分析與 CSV 匯出。

## 功能特色

- 自動顯示今日星期（週一～週日）
- 動態產生每日課表
- 支援三種記錄模式：
  - **打卡模式**：簽到 / 簽退 / 停留時間（分鐘）
  - **學習檢核模式**：完成、未玩鬧、遵守 三項勾選
  - **自主活動模式**（特定節次）：活動選擇與回饋提示
- 午餐時段簽到提醒用藥
- 一鍵產生「今日分析」
- 一鍵匯出 CSV（`今日紀錄.csv`）

## 使用方式

1. 下載或複製專案後，直接用瀏覽器開啟 `index.html`
2. 輸入學生姓名（選填）
3. 依照各節次進行記錄：
   - 打卡節次：按「簽到」、「簽退」
   - 學習節次：勾選三項檢核後按「✔」
   - 自主活動節次：選擇活動後按「✔」
4. 需要統整時按：
   - `📊 今日分析`：查看平均停留與異常次數
   - `📥 匯出紀錄`：下載 CSV 檔

## 資料欄位說明（CSV）

- `類型`：記錄類別（學習/活動/打卡）
- `簽到`：簽到時間
- `簽退`：簽退時間
- `停留時間`：簽到到簽退的分鐘差
- `內容`：活動內容（如：寫評量、晤談）

## 專案結構

- `index.html`：主程式（HTML + CSS + JavaScript）
- `README.md`：專案說明文件

## 技術說明

- 無框架，原生 JavaScript
- 不需後端與資料庫
- 資料暫存在瀏覽器記憶體，重新整理頁面會清空

## 已知限制

- 目前沒有長期儲存（LocalStorage / 雲端）
- 統計邏輯可再加強（例如未簽退判斷一致性）
- 時間差計算以本地時間字串轉換為主，未處理跨日情境

## 後續可擴充方向
